# cien-downloader_chromeExtension
ci-enの動画/音声をダウンロードするためのchrome拡張機能（実態はHTMLからurlを抽出してダウンロードさせてるだけ）

## 導入方法
### 1. chrome系ウェブブラウザを開きます(Google Chrome, Microsoft Edge, Chromium, Vivaldiなど)
### 2. 拡張機能のページを開きます
 - Google Chromeの場合: ```chrome://extensions```
 - Microsoft Edgeの場合: ```edge://extensions```
 - Vivaldiの場合: ```vivaldi://extensions```
<img src="https://github.com/user-attachments/assets/5bd93bc4-270a-4035-adac-39683da5b0bd" width="70%">

### 3. 右上の「デベロッパーモード」をONにします
<img src="https://github.com/user-attachments/assets/941491dc-da1f-4491-a133-9c286f003708" width="70%">

### 4. [最新リリース](https://github.com/chcolte/cien-downloader_chromeExtension/releases/latest) から"cien-downloader_chromeExtension-main.zip"をダウンロードし，展開します。

### 5.「パッケージ化されていない拡張機能を読み込む」ボタンをクリックし，ダウンロードした「フォルダ」を選択します。
<img src="https://github.com/user-attachments/assets/7fa6738f-b0b2-468d-bcbb-38d432eb8446" width="70%">

### 6. 拡張機能がインストールされます
<img src="https://github.com/user-attachments/assets/0a202ddf-8abc-4efc-b97f-b7a1cb09b040" width="70%">

### 7. 有効化します
📌マークを押してピンどめもできます。

<img src="https://github.com/user-attachments/assets/2450674a-b215-47e8-85ca-22463ed81109" width="70%">

### 8. 使用する
- ファイル名をクリックで，そのファイル単体をダウンロード
- 「すべてダウンロード」で，すべてのファイルをダウンロード
